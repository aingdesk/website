# 分享

## 创建分享链接
1. 点击界面右上角的分享按钮。
![](img/2025-02-20_142329_202.png)
2. 选择模型，设置名称，并点击分享按钮。
![](img/2025-02-20_142517_948.png)
3. 等待生成分享链接后，即可复制链接分享给好友使用。
![](img/2025-02-20_142617_717.png)

## 使用分享模型
1. 点击链接，进入分享页面。
2. 输入对话内容，点击发送按钮即可与模型对话。
![](img/2025-02-20_142753_623.png)
