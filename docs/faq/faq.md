# 常见问题
## AingDesk是不是永久开源免费的？

AingDesk是MIT开源协议软件，个人公司都可无限制的免费使用。

## AingDesk需要什么最低配置的电脑安装？
Windows/Linux :

CPU: Intel i5 / AMD Ryzen 5

GPU: 推荐带有NVIADIA独立显卡

内存: 8GB

硬盘: 10GB SSD

macOS (Apple Silicon) :

芯片: M1 或更高

内存: 8GB

硬盘: 10GB

## 部署DeepSeek的大概操作步骤是什么？
1.根据自己电脑系统下载安装AingDesk
2.按照提示安装模型运行软件
3.选择相应的DeepSeek模型安装即可
