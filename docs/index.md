# 介绍
## 🚀 一句话简介
简单好用的 AI 助手软件，支持本地 AI 模型及 API + 知识库搭建。

## ✅ 核心功能
* 一键部署本地 AI 模型及主流模型 API
* 本地知识库
* 智能体创建
* 可在线分享给他人使用
* 支持联网搜索
* 支持服务器端部署
* 支持 MCP
* 单次多模型同时对话（即将上线）

## ✨ 产品亮点
* 简单好用，对 AI 新手友好
